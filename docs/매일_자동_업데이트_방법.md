# 매일 정해진 시간에 프로필 잔디 자동 업데이트 — 딱 2단계

지금 프로필에 잔디가 뜨는 건 맞게 된 거예요.  
**매일 새벽 1시에 저절로 갱신**되게 하려면 아래만 하면 됩니다.

---

## 1단계: 토큰 만들기 (1분)

1. GitHub **오른쪽 위 프로필 사진** 클릭 → **Settings**
2. 왼쪽 맨 아래 **Developer settings** 클릭
3. **Personal access tokens** → **Tokens (classic)** → **Generate new token (classic)**
4. **Note**에 아무 이름 (예: `잔디자동`)
5. **Expiration**에서 기간 선택 (90 days 또는 No expiration)
6. **repo**에 체크
7. **Generate token** 누르고 **나온 토큰 복사** (한 번만 보여주니까 꼭 복사)

---

## 2단계: green-movement 저장소에 토큰 넣기 (1분)

1. **green-movement** 저장소 페이지로 이동 (내가 만든 잔디 프로젝트)
2. **Settings** → 왼쪽에서 **Secrets and variables** → **Actions**
3. **New repository secret** 클릭
4. **Name**: `PROFILE_README_TOKEN` 그대로 입력
5. **Secret**: 아까 복사한 토큰 붙여넣기
6. **Add secret** 클릭

---

## 끝

이렇게만 해두면 **매일 새벽 1시(KST)**에 GitHub가 알아서:

- 잔디 SVG 만들고
- 프로필 저장소(SeoNaRu)의 `assets/live.svg` 갱신

해서 프로필에 보이는 잔디가 하루에 한 번씩 자동으로 업데이트됩니다.

- **시간 바꾸고 싶으면**: `.github/workflows/update-profile-readme.yml` 들어가서 `cron: "0 16 * * *"` 숫자만 바꾸면 됨 (자세한 건 PROFILE_README_SETUP.md 참고)
- **지금 당장 한 번 돌려보고 싶으면**: green-movement 저장소 → **Actions** 탭 → **Update profile README with grass SVG** → **Run workflow**
